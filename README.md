# online_floral_and_gift_management_system# 🌸 Online Floral & Gift Management System

## 📌 Introduction
This project is a web-based application developed to manage floral and gift products online. It allows users to browse products, add to cart, and place orders.

## 🚀 Features
- User authentication (Login/Register)
- Product listing and categories
- Add to cart
- Checkout system
- Admin dashboard for product management

## 🛠️ Technologies Used
- Frontend: React.js
- Backend: Laravel
- Database: MySQL
- API: RESTful API
